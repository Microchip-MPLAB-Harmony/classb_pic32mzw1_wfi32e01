# CLASSB\_CLOCK\_MAX\_CLOCK\_FREQ

**Summary**

Maximum CPU clock speed.

**Description**

Defines the maximum CPU clock speed for the microcontroller.

**Remarks**

This value must not be modified.

```c
#define CLASSB_CLOCK_MAX_CLOCK_FREQ (200000000U)
```
